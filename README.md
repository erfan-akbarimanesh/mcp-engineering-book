# mcp-engineering-book
# مقدمه

در چند سال گذشته مدل‌های زبانی بزرگ یا همان LLM ها شیوه تعامل ما با نرم‌افزار را تغییر داده‌اند. اگر چند سال قبل ساخت یک سیستم نرم‌افزاری به معنی طراحی API ها، پایگاه داده و سرویس‌های Backend بود، امروز باید یک لایه جدید نیز به آن اضافه کنیم، لایه تعامل مدل‌های هوش مصنوعی با جهان واقعی.

مدل‌های زبانی ذاتا در یک محیط بسته کار می‌کنند. آن‌ها به اینترنت دسترسی ندارند، نمی‌توانند مستقیماً به پایگاه داده متصل شوند و قادر نیستند ابزارهای خارجی را اجرا کنند. در نتیجه اگر بخواهیم از آن‌ها برای ساخت سیستم‌های واقعی استفاده کنیم، باید راهی برای اتصال امن و ساختاریافته آن‌ها به ابزارها و منابع داده ایجاد کنیم.

**Model Context Protocol** یا **MCP** دقیقاً برای حل همین مسئله طراحی شده است. این پروتکل یک استاندارد برای اتصال مدل‌های هوش مصنوعی به ابزارها، منابع داده و سرویس‌های خارجی فراهم می‌کند.

## هدف کتاب
این کتاب برای مهندسان نرم‌افزار نوشته شده است؛ افرادی که می‌خواهند فراتر از استفاده ساده از مدل‌های هوش مصنوعی حرکت کنند و سیستم‌های واقعی مبتنی بر AI بسازند. در این کتاب تلاش شده است MCP نه صرفاً به عنوان یک ابزار، بلکه به عنوان بخشی از معماری سیستم‌های هوش مصنوعی بررسی شود.

هدف این کتاب ارائه یک دید مهندسی نسبت به MCP است. به جای تمرکز صرف بر مثال‌های ساده یا توضیح مفاهیم سطحی، سعی شده است نحوه استفاده از این پروتکل در طراحی سیستم‌های واقعی بررسی شود. مسائلی مانند طراحی ابزارها، مدیریت context، ارتباط بین agent ها و نحوه سازمان‌دهی سرویس‌های AI از جمله موضوعاتی هستند که در ادامه به آن‌ها پرداخته خواهد شد.

## مخاطبان هدف
مخاطب این کتاب برنامه‌نویسان و معماران نرم‌افزار هستند؛ افرادی که با مفاهیم پایه توسعه نرم‌افزار، API ها و سیستم‌های Backend آشنا هستند و می‌خواهند درک عمیق‌تری از نحوه ساخت سیستم‌های مبتنی بر مدل‌های زبانی به دست آورند.

## نقشه راه (فصل‌های کتاب)
در فصل‌های پیش رو ابتدا با مفاهیم پایه MCP آشنا می‌شویم، سپس:
*   نحوه ساخت یک **MCP Server** را بررسی می‌کنیم.
*   به سراغ طراحی ابزارها، اتصال به مدل‌های زبانی و معماری سیستم‌های پیچیده‌تر مبتنی با MCP خواهیم رفت.
*   در بخش‌های پایانی نیز به چالش‌های عملیاتی، استقرار در محیط **production** و طراحی اکوسیستم ابزارها پرداخته می‌شود.
